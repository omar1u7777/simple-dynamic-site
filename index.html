Välkommen

Denna enkla dynamiska webbplats visar inlägg, användarprofiler, kommentarer och ett kontaktformulär. Gå till sidan "Inlägg" för att bläddra och klicka på ett användarnamn för att se profilen.

Instruktioner: Gå till "Inlägg" för att ladda inlägg. På kontaktsidan fyller du i formuläret och bekräftar för att kunna skicka.

Inlägg

Dynamiska inlägg hämtade från ett API. Varje inlägg visar titel, innehåll, taggar, reaktioner och kommentarer.

Profiler

Klicka på ett användarnamn för att öppna en modal med profilinformation (namn, e-post, adress och mer).

Kontakt

Enkelt validerat kontaktformulär för att skicka meddelande.
